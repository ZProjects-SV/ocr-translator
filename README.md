## 🇬🇧 English

A desktop application for Windows that lets you select any area of the screen, extract text using OCR (Optical Character Recognition), and translate it in real time. Built with Python, PySide6 (Qt), PaddleOCR, and deep-translator.

### 📦 Available Versions

| Version | Description | Folder |
| :--- | :--- | :--- |
| 🏗️ **Source Code** | For developers. Clone, modify and build your own version. | [`source/`](./source/) |
| 📂 **Portable Executable** | No installation needed. Download, extract and run. | [`executable/`](./executable/) |
| 💿 **Installer (Setup)** | Traditional Windows installer with shortcuts. | [`installer/`](./installer/) |

### ✨ Features

- **Interactive capture** — Select any screen region with a precision cursor.
- **Advanced OCR** — PaddleOCR with image preprocessing for higher accuracy.
- **Real-time translation** — Google Translate integration with cache system.
- **System Tray** — Runs silently in the background.
- **Custom hotkeys** — Configure your own keyboard shortcuts.
- **Language downloads** — Download extra OCR models from the Preferences window.

## 🇸🇻 Español

Una aplicación de escritorio para Windows que permite seleccionar cualquier área de la pantalla, extraer el texto mediante OCR y traducirlo en tiempo real. Construida con Python, PySide6 (Qt), PaddleOCR y deep-translator.

### 📦 Versiones disponibles

| Versión | Descripción | Carpeta |
| :--- | :--- | :--- |
| 🏗️ **Código Fuente** | Para desarrolladores. Clona, modifica y compila tu propia versión. | [`source/`](./source/) |
| 📂 **Ejecutable Portable** | Sin instalación. Descarga, extrae y ejecuta. | [`executable/`](./executable/) |
| 💿 **Instalador (Setup)** | Asistente de instalación tradicional con accesos directos. | [`installer/`](./installer/) |

### ✨ Características

- **Captura interactiva** — Selecciona cualquier área de la pantalla con cursor de alta precisión.
- **OCR Avanzado** — PaddleOCR con preprocesamiento de imagen para mayor precisión.
- **Traducción en tiempo real** — Integración con Google Translate y sistema de caché.
- **System Tray** — Se ejecuta en segundo plano en la bandeja del sistema.
- **Atajos personalizables** — Configura tus propios atajos de teclado.
- **Descarga de idiomas** — Descarga modelos OCR adicionales desde Preferencias.
