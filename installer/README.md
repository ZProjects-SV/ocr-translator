## 🇬🇧 English

Traditional Windows installer. Creates Start Menu and Desktop shortcuts and allows clean uninstallation from Control Panel.

### Installation Steps

1. **Download** `OCR_Translator_Setup.exe` from the [Releases](https://github.com/ZProjects-SV/ocr-translator/releases) page.
2. **Run** the downloaded file.
3. If Windows SmartScreen appears → click **"More info"** → **"Run anyway"** (not digitally signed, but source code is fully public).
4. Follow the setup wizard:
   - Accept the license agreement (GPL v3).
   - Choose destination folder (default: `C:\Program Files\OCR Translator`).
   - Choose whether to create Desktop and Start Menu shortcuts.
5. Click **"Install"**, then **"Finish"**.

You can now find **OCR Translator** in your Start Menu.

### Uninstalling

**Option 1 — Settings:**
> Settings → Apps → OCR Translator → Uninstall

**Option 2 — Control Panel:**
> Control Panel → Programs → Programs and Features → right-click OCR Translator → Uninstall

> 💡 Your saved preferences and custom hotkeys are stored in the Windows Registry and are **not** removed on uninstall — they persist if you reinstall later.

### Default Installation Path

```
C:\Program Files\OCR Translator\
├── OCR_Translator.exe
├── _internal\          ← Do not delete
└── uninstall.exe
```

## 🇸🇻 Español

Instalador tradicional de Windows. Crea accesos directos en el Menú de Inicio y Escritorio, y permite una desinstalación limpia desde el Panel de Control.

### Pasos de instalación

1. **Descarga** `OCR_Translator_Setup.exe` desde la página de [Releases](https://github.com/ZProjects-SV/ocr-translator/releases).
2. **Ejecuta** el archivo descargado.
3. Si Windows SmartScreen aparece → haz clic en **"Más información"** → **"Ejecutar de todas formas"** (sin firma digital, pero el código fuente es completamente público).
4. Sigue el asistente:
   - Acepta el acuerdo de licencia (GPL v3).
   - Elige la carpeta de destino (por defecto: `C:\Program Files\OCR Translator`).
   - Elige si deseas acceso directo en el Escritorio y Menú de Inicio.
5. Haz clic en **"Instalar"** y luego en **"Finalizar"**.

Ya puedes encontrar **OCR Translator** en tu Menú de Inicio.

### Desinstalar

**Opción 1 — Configuración:**
> Configuración → Aplicaciones → OCR Translator → Desinstalar

**Opción 2 — Panel de Control:**
> Panel de Control → Programas → Programas y características → clic derecho en OCR Translator → Desinstalar

> 💡 Tus preferencias y atajos personalizados se almacenan en el Registro de Windows y **no se eliminan** al desinstalar, por si decides reinstalar en el futuro.

### Ruta de instalación por defecto

```
C:\Program Files\OCR Translator\
├── OCR_Translator.exe
├── _internal\          ← No eliminar
└── uninstall.exe
```
